# /examples — תמונות רפרנס של Atla Batla

פליירים, ארטוורק ריליסים וצילומי מסך אמיתיים של Atla Batla.

**כל החלטת עיצוב חייבת להיגזר מהתמונות כאן** — פלטה, טיפוגרפיה, מוטיבים.
אסור לנחש אסתטיקת "גואה-טראנס" גנרית (PRD §2).

Claude Code חייב `view` על כל קובץ בתיקייה **לפני** קביעת design tokens
(זה קורה ב-Phase 2). התיקייה ממולאת ידנית ע"י Sisu.

המוטיבים שכבר זוהו בפליירים האמיתיים: שמיים שחורים זרועי כוכבים, איורי AI
מקסימליסטיים וצבעוניים, מסקוט גמל הומוריסטי, "אוטובוס מפלצת", lineup בגרדיאנט
rainbow. גם משמשות כתמונות רפרנס ל-pipeline של Gemini (PRD §7).
