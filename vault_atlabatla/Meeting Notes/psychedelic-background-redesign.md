# Psychedelic Background Redesign

## Overview
עיצוב מחדש של רקע האמצע+תחתית של עמוד הנחיתה בסגנון עטיפת האלבום (`examples/background/11277.jpg` ושאר ההשראות ב-`examples/background/`): שמי שקיעת-מדבר רציפים — לילה → סגול → מג'נטה → כתום → זוהר-זהב באופק → אדמת דמדומים. שתי מטרות מהמשתמשת: (1) האמצע+תחתית "בסגנון הזה"; (2) **מעבר גרדיאנט חלק מה-Hero אל שאר הדף — בלי "חיתוך"**. הבעיה המקורית: ה-Hero דוהה ל-`--night` אטום בתחתיתו, ומיד מתחתיו נחשפת שכבת ה-`.aurora` הגלובלית (fixed, בועות violet/magenta במיקומי-viewport קבועים) → בליטת-צבע פתאומית = ה"חיתוך". חי כרגע **בטיוטה בלבד** תחת `app/draft/background/` (gitignored, לא קודם). 3 קבצים: `psychedelic.css` (מחלקות `pb-*`), `PsychedelicBackdrop.tsx` (שכבת רקע דקורטיבית), `page.tsx` (fork של `app/page.tsx`). תצוגה: `/draft/background`.

- קידום: כרגע ה-footer מוכנס *לתוך* `main`/`.pb-region` לרציפות רקע; קידום ל-`app/page.tsx` האמיתי ידרוש החלטה — לשכתב את מבנה העמוד או להרחיב את הרקע מאחורי ה-footer בלי להזיז אותו. הרקע החדש גם צריך לגבור/להחליף את ה-`.aurora` הגלובלי (מנוטרל בטיוטה ע"י כיסוי אטום; בפרודקשן אולי להסיר את `<AuroraBackground/>` מה-layout או להתנות אותו).

### 2026-07-03 — טיוטת רקע שקיעת-מדבר + מעבר חלק מה-Hero [spiked]
- **What was done:** נבנתה טיוטה ב-`app/draft/background/` (gitignored): `psychedelic.css` — שכבת `.pb-backdrop` (z-index:-20, absolute) עם `.pb-sky` (גרדיאנט אנכי אטום שהפס העליון שלו == `#0b0420`=`--night` כדי להתמזג עם ה-Hero), `.pb-sun`+`.pb-rays` (שמש-זהב + קרני-ספירלה מסתובבות איטית, ממוסכות), `.pb-blob-1/2/3` (magenta/sunset/violet מרחפים, blur+screen), `.pb-stars` (רק בשמי הלילה העליונים, מסכה), `.pb-vignette` (מחשיך שוליים לקריאוּת), `.pb-seam` (night→שקוף על גבול ה-Hero). `PsychedelicBackdrop.tsx` — קומפוננטת שרת דקורטיבית שמרנדרת את השכבות. `page.tsx` — fork של `app/page.tsx`: `SiteHeader`+`Hero` מיובאים מהמקור **ללא נגיעה**; כל מה שאחריהם עטוף ב-`.pb-region` (relative+isolate) עם ה-backdrop, כולל ה-`SiteFooter` (הוכנס ל-`main` כדי שהרקע יימשך עד התחתית).
- **Decisions:** (1) רקע כשכבת `absolute` אטומה בתוך `.pb-region` עם `isolation:isolate` — מכסה את ה-`.aurora` הגלובלי (fixed z-index:-1) שנשאר מה-root layout, בלי צורך לגעת ב-`globals.css`/`layout.tsx`. (2) הגרדיאנט ממופה על אזור האמצע+תחתית בלבד (0%=מתחת ל-Hero=לילה, 100%=footer=דמדומים) → "סיפור" אנכי של שקיעה + תפר-לילה עליון = **בלי חיתוך**, robust לגובה-תוכן משתנה. (3) `.pb-seam` (30svh) כביטוח כפול על גבול ה-Hero. (4) קידומת `pb-` + CSS מיובא רק ב-route של הטיוטה → אפס התנגשות עם האתר האמיתי. (5) שימוש בטוקני מותג (`--magenta`/`--sunset`/`--violet`/`--night`) מ-`:root` הגלובלי. (6) `MovingObjects` נשמר, עטוף ב-wrapper פנימי `relative isolate` כדי שיישאר מוגבל לאמצע+תחתית כמו בפרודקשן.
- **Notes / Caveats:** אומת ב-curl (ה-Preview renderer לא אמין בסביבה — כמו ב-sessions קודמים): `/draft/background` → HTTP 200; כל שכבות ה-`pb-*` נוכחות ב-HTML; ה-Hero + הסקשנים האמיתיים (music/about/booking/footer) מרונדרים; ה-CSS chunk `app_draft_background_psychedelic_*.css` מוגש עם כללי ה-pb; אפס שגיאות ריצה. הוצג למשתמשת mock ויזואלי (SVG דרך visualize) שמשקף את *אותם* עצירות-גרדיאנט. **לא קודם לפרודקשן** — ממתין לאישור מפורש ("לדחוף לורסל"). `next dev` הורץ ידנית על port 3999 (ה-`preview_start` נפל מיד — infra לא יציב פה).
