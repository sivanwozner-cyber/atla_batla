# Psychedelic Background Redesign

## Overview
עיצוב מחדש של רקע האמצע+תחתית של עמוד הנחיתה בסגנון עטיפת האלבום (`examples/background/11277.jpg` ושאר ההשראות ב-`examples/background/`): שמי שקיעת-מדבר רציפים — לילה → סגול → מג'נטה → כתום → זוהר-זהב באופק → אדמת דמדומים. שתי מטרות מהמשתמשת: (1) האמצע+תחתית "בסגנון הזה"; (2) **מעבר גרדיאנט חלק מה-Hero אל שאר הדף — בלי "חיתוך"**. הבעיה המקורית: ה-Hero דוהה ל-`--night` אטום בתחתיתו, ומיד מתחתיו נחשפת שכבת ה-`.aurora` הגלובלית (fixed, בועות violet/magenta במיקומי-viewport קבועים) → בליטת-צבע פתאומית = ה"חיתוך". **v1 (גרדיאנט מסונתז) קודם לפרודקשן ונדחף** (commit `a2d4e1d` ב-origin/main → Vercel): ה-`pb-*` ב-`app/globals.css`, `components/PsychedelicBackground.tsx`, `app/page.tsx` עוטף אמצע+תחתית+footer ב-`.pb-region`, ו-`<AuroraBackground/>` הוסר מ-`app/layout.tsx`. **v2** (`app/draft/background-v2/`, בטיוטה) — **עבר איטרציה**: התחיל כתמונת עטיפת-האלבום (`11277.jpg`) מלאה (המשתמשת פסלה — "עמוס"), ועכשיו = מבני-סלע דהויים בצדדים בלבד (רפרנס: ארטוורק עמק-מקדשים ששלחה) בצבעי-המותג, אמצע פתוח, CSS/SVG טהור (`scenery.css` + `SceneryBackdrop.tsx`, בלי תמונה חיצונית). **v3** (`app/draft/background-v3/`, בטיוטה) — תמונה שלישית שהמשתמשת שלחה (`examples/background/V3/…jpeg`, מבני-מקדש/טוטם מצוירים בצדדים + מסדרון-שמיים פתוח באמצע — קרוב מאוד לקומפוזיציה של v2 אך כתמונה מלאה). **עודכן ל-Gemini-adapted asset**: המשתמשת ביקשה "התאמה עם ג'ימיני להירו הקיים" — הופק `public/generated/bg-v3-gemini.png` (3:4, 2K, דרך `scripts/generate-art.mjs` עם התמונה כ-ref) ששומר את הקומפוזיציה כמעט זהה אך צובע מחדש את הפס העליון ל-`#0b0420`=`--night` עם כוכבים עדינים בלבד (בלי גלקסיות/צבע עז שם) — מיזוג נאמן יותר עם ה-Hero. זו התמונה שבפועל ב-`ImageBackdrop.tsx` (לא ה-jpg הגולמי, שנמחק אחרי השימוש). scrim (עמודת-מרכז כהה לקריאוּת, צדדים בהירים לחשוף מבנים) + דהיית-לילה עליונה + seam (DOM-relative, לא תלוי בתוכן התמונה) נשארו כביטוח. תצוגה: `/draft/background` (fork v1 מקורי), `/draft/background-v2` (v2), `/draft/background-v3` (v3, Gemini-adapted).
